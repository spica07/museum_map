# 전국 박물관·미술관 지도

전국 박물관·미술관 505곳을 지도에서 찾아보는 정적 웹앱입니다. `index.html` 한 장으로 동작하며 빌드 과정이 없습니다.

## 데이터 출처

- 공공데이터포털 「전국박물관미술관정보표준데이터」(publicDataPk=15017323)
- 좌표는 주소 기반 카카오 로컬 API 지오코딩으로 보완
- 운영시간·요금 등은 조사 시점 기준이며 방문 전 재확인이 필요합니다.

## 기술 스택

- Leaflet + OpenStreetMap 타일
- 빌드 없는 정적 HTML/CSS/JS, GitHub Pages(main 브랜치) 배포
